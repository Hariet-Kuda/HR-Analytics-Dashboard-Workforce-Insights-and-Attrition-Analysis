**HR Analytics Dashboard (Power BI)**  
*Overview*  

This project presents an interactive HR analytics dashboard built in Power BI to provide insights into workforce trends, employee demographics, and attrition patterns.
The goal is to support HR and leadership teams in making data-driven decisions around employee retention, hiring, and workforce planning.

*Key Business Questions*
1. What is the overall employee attrition rate?
2. Which departments or roles have the highest turnover?
3. How do factors like age, tenure, and salary impact attrition?
4. What are the workforce demographics (gender, age distribution, etc.)?

*Tools & Technologies*  
-Power BI (Data Modeling & Visualization)  
-DAX (Calculated Measures)  
-Power Query (Data Transformation)  

*Project Structure*  
-report/ → Dashboard visuals and layout  
-semantic model/ → Data model, relationships, DAX measures  
-.pbip → Power BI project file
